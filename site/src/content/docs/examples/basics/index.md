Nine examples, one core principle each. Read them in order or jump straight to the API you want to see.

Every snippet is **complete and copy-pasteable** into a Vite project with `kerfjs` installed and `tsconfig.json` configured for `"jsxImportSource": "kerfjs"` (see [Install](/kerf/#install)).

| 3 | [Store](/kerf/examples/basics/03-store/) | `defineStore` + named actions, `reset()` |
| 4 | [Mount + delegate](/kerf/examples/basics/04-mount-delegate/) | `delegate` + `data-action` |
